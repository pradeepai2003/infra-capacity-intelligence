InfraCapacityDashboard.pbix is a binary Power BI Desktop file and can't be
generated by a script. To create it:

1. Run the pipeline once (`python -m src.pipeline.run_pipeline`) so the CSVs
   in this folder are populated: utilization_overview.csv, recommendations.csv,
   risk_summary.csv.
2. Open Power BI Desktop -> Get Data -> Folder -> select src/powerbi/.
3. Build the visuals described in refresh_config.json under "recommended_visuals".
4. Save as InfraCapacityDashboard.pbix in this folder.
5. Commit the .pbix file (Git treats it as a binary blob - consider Git LFS
   if it grows large).
